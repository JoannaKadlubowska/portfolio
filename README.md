# O mnie
Obecnie zdobywam komercyjne doświadczenie zajmując się testowaniem aplikacji mobilnej przeznaczonej do realizacji zakupów w handelu detalicznych branży spożywczej.
      Lubię stawiać sobie wyzwania i  dążyć do ich realizacji. Rozwój jest dla mnie  ważny.  Jestem otwarta na zdobywanie  nowej wiedzy i rozwijanie umiejętności technicznych w dziedzinie testowania oprogramowania.
